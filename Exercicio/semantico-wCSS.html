DIVs e HTML semantico

As div são divisorias que o site pode ter, a primeira vista ela não parece ter nada de diferente se comparado com o HTML feito normalmente mas com o CSS, as divs viram outra coisa.

Para adicionar o CSS a essas divs, é necessario atribuir uma classe a cada uma e adicionar no head a tag style que é onde serão feitas as alterações.
