Pulando linhas

Para pular linhas, é utilizado a tag br que se colocada no meio de um paragráfo, a partir da frente da tag será colocada uma linha abaixo, olhe o exemplo abaixo:

Lorem ipsum dolor sit amet consectetur adipisicing elit. Ipsa est sed iusto, ipsam id vel? Sed nostrum nesciunt ab ducimus sequi veritatis reiciendis.
Molestiae officia expedita iste voluptatum quo voluptate?

A tag br está antes de "molestiae", logo, apartir de Molestiae será uma nova linha. Caso seja necessário pular uma linha, basta colocar o br novamente.

Endereços e citações

Endereço

Caso você vá colocar um endereço de um lugar ou um endereço eletrônico, utilize a tag address, ela deixará o texto em italico e formatado.

Citações

Quando você for citar uma obra artistica ou uma produção, use a tag cite, ela já deixará bem formatado. Mas se você for citar uma fala direta de um personagem ou de alguém, use a tag q, ela já deixara com as aspas e tudo bem formatado.

Veja esta fala presente na obra Kit salve da quebrada: Um jacarezinho para amarrar as mãozinhas dela ah sim, um taco de baseball para quebrar os braços dela ah sim.

Listas

Listas podem ser utilizadas para colocar varios itens em sequencia dentro do sitee existem dois tipos de listas:

  • Lista desordenada
  • Lista ordenada

Lista desordenada

As listas desordenadas tão tem ordem, elas apresentam os itens de levar em consideração quem veio primeiro, segundo e etc. Ela demonstra eles de forma normal colocando apenas um ponto como pode ser observado no exeplo a seguir:

  • Banana
  • Maça
  • Peira
  • Uva

Lista ordenada

As listas ordenadas colocam a posição do item, como por exemplo:

  1. Eu
  2. Tu
  3. Ele

Para as listas serem utilizadas, a tag ul é utilizada para inicar a lista desordenada enquanto para iniciar a lista ordenada é utilizada a tag ol e dentro de ambas as tags vai a tag li que siginfica list item.

Formularios

Os formularios servem para deixar o usuario dar input de alguns dados a serem utilizados ou guardados no backend.

Um exemplo pode ser visualizado na pagina a seguir:

Formulario

O formulario é iniciado pela tag form que é a tag "pai" que tem diversas outras tag que vão dentro dela para recolher todo tipo de dado que seja necessario.

Para isso, existe a tag input que representa a entrada de um dado e esse input tem diversos tipos de dados como:

INPUT

tag uso
text Entrada de texto simples
password Quando digitado algo, os caracteres ficam na forma de asterisco *
email Ele reconhece se o que o usuario escreveu é um email ou não
submit Botão enviar dados
radio É utilizado para dar varias opções marcaveis para o usuario mas apenas uma pode ser selecionada
checkbox É utilizado para dar varias opções marcaveis para o usuario e ele pode selecionar mais de uma opção

Alem disso, existe também o item placeholder que é um texto meio apagado que pode ser uma dica do que o usuario deve escrever naquele campo. Assim como também existe o "label" que é um texto que fica antes do input. Outro atributo do input é que ele pode ser um campo obrigatorio a ser preenchido ao colocar o atributo required


SELECT

O select é utilizado para escolher uma opção entre outras demais disponiveis em um campo de seleção, um exemplo desse campo é quando tem que escolher o estado que você mora e na lista tem o nome de todos os estados mas você escolhe apenas um.


TEXT AREA

O TEXT AREA é um campo para que o usuario possa escrever o que quiser, sendo um campo grande assim como em sites tem um campo para fazer comentarios, esse campo é feito justamente com o text area
